# Casos de Uso

BSP é infraestrutura. Estas páginas mostram o que se constrói sobre ela.

## [Soberania de Dados de Saúde](/pt/use-cases/health-data-ownership)

Tome posse permanente dos seus exames de sangue, registros médicos e dados de wearables. Sem intermediários. Sem expiração. Ancorado no Arweave para sempre.

---

## [Registros de Saúde Descentralizados](/pt/use-cases/decentralized-health-records)

Registros de saúde permanentes e verificáveis criptograficamente — sem depender de nenhuma instituição permanecer online. Construídos sobre Arweave + processos AO.

---

## [Padrão Aberto de Dados de Saúde](/pt/use-cases/open-health-data-standard)

Um padrão aberto orientado a desenvolvedores para dados biológicos. Comparação entre BSP, FHIR, HL7 e APIs proprietárias. Feito para labs, clínicas e plataformas de saúde.

---

## [Dados para IA de Longevidade](/pt/use-cases/longevity-ai-data)

Dados biológicos estruturados, tipados e verificados criptograficamente para engines de IA de longevidade, análise de biomarcadores e modelos de medicina de precisão.
