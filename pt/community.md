# Junte-se à Comunidade

## Bem-vindo à comunidade BSP

A comunidade BSP é uma rede crescente de desenvolvedores, pesquisadores, profissionais de saúde e indivíduos que acreditam que a soberania biológica é um direito humano fundamental.

## Por que se envolver?

### Construa o que importa

O BSP é infraestrutura para os dados mais pessoais que existem. Toda contribuição — código, documentação, tradução, feedback — avança a autonomia humana.

### Molde o protocolo

Qualquer pessoa pode propor mudanças via BIPs (BSP Improvement Proposals). O roadmap do protocolo é aberto. Sua expertise em genômica, criptografia, saúde ou UX pode melhorar diretamente o BSP.

### Conecte-se com uma comunidade rara

Pessoas trabalhando na interseção de protocolos abertos, privacidade, longevidade e dados biológicos são raras. O BSP as atrai.

## Como se envolver

### Contribua para o protocolo

Leia issues abertas, proponha BIPs ou revise propostas existentes. Todo trabalho de especificação acontece de forma aberta.

[Ver GitHub](https://github.com/biological-sovereignty-protocol) · [Issues Abertas](https://github.com/biological-sovereignty-protocol/bsp-spec/issues)

### Construa sobre o BSP

Use o SDK TypeScript ou Python para construir apps de saúde, ferramentas de pesquisa ou integrações de dados sobre o protocolo.

[Docs para Desenvolvedores](/pt/developers/sdk-reference) · [Quickstart](/pt/getting-started/quickstart)

### Contribua com documentação

Traduções, melhorias, novos guias — contribuições de documentação são sempre bem-vindas.

[Guia de Contribuição](https://github.com/biological-sovereignty-protocol/bsp-spec/blob/main/CONTRIBUTING.md)

### Proponha um BIP

Tem uma ideia para melhorar o BSP? Escreva um BIP. A comunidade revisa e vota em todas as mudanças do protocolo.

[Ver BIPs](/pt/bips/) · [Enviar um BIP](https://github.com/biological-sovereignty-protocol/bsp-spec/issues/new)

## Construído pela comunidade

O BSP é mantido por contribuidores de pesquisa, saúde, criptografia e engenharia de software. Nenhuma empresa controla o roadmap.

[Ver contribuidores no GitHub](https://github.com/biological-sovereignty-protocol)
