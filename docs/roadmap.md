# BSP Roadmap

The path to universal biological sovereignty.

## Completed

### Phase 0 — Foundation
**2024 Q3**

- BEO and IEO specification drafted
- Ed25519 signature scheme defined
- Arweave storage model designed
- Initial SmartWeave contract prototypes

### Phase 1 — Core Protocol v1
**2025 Q1**

- BEO registration and management contracts deployed on Arweave
- IEO registry contracts deployed
- ConsentToken issuance and revocation
- BSP Registry API (Node.js) — public alpha

### Phase 2 — SDK & Developer Tools
**2025 Q2–Q3**

- TypeScript SDK (`@bsp/sdk`) published to npm
- Python SDK (`bsp-sdk`) published to PyPI
- CLI tooling for BEO management
- BSP documentation site launched
- Exchange API for BioRecord submission

### Phase 3 — Specification Freeze
**2025 Q4**

- BSP Specification v1.0 finalized
- BIP process (BSP Improvement Proposals) established
- Security audit completed
- BSP v1.0 mainnet deployment

## In Development

### Phase 4 — Ecosystem Expansion
**2026**

- Wearable device integrations (Apple Health, Garmin, Oura)
- Genomics lab connectors (23andMe, Nebula, Sequencing.com)
- BSP Identity mobile app
- HealthRecord standard (BioRecord v2)

### Phase 5 — Governance & Decentralization
**2026–2027**

- On-chain BIP voting mechanism
- Multi-sig protocol upgrades
- Decentralized registry nodes
- Cross-chain BSP anchors (Ethereum, Solana)

### Phase 6 — AI Data Markets
**2027+**

- Consented biological data marketplace
- AI training dataset pools (opt-in)
- Longitudinal cohort data products
- Revenue sharing for data contributors
