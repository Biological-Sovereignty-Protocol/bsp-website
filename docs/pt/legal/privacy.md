# Política de Privacidade

**Última atualização:** Março de 2026
**Publicado por:** Instituto Ambrósio

---

## Visão Geral

O Protocolo de Soberania Biológica (BSP) é um padrão aberto. Esta Política de Privacidade se aplica ao site BSP (`biologicalsovereigntyprotocol.com`) e sua documentação.

O BSP não opera uma plataforma centralizada de dados. Os dados biológicos enviados pelo protocolo são armazenados diretamente na rede Arweave — não em servidores controlados pelo Instituto Ambrósio.

---

## Informações que Coletamos

**Analytics do site:** Podemos coletar dados de uso anonimizados (visualizações de página, região geográfica, tipo de navegador) para melhorar a documentação. Nenhuma informação pessoal identificável é coletada.

**Contato:** Se você nos contatar via GitHub Issues ou e-mail, mantemos essa correspondência para responder e melhorar o protocolo.

---

## O que Não Fazemos

- Não vendemos, compartilhamos nem monetizamos dados de usuários.
- Não rastreamos indivíduos em outros sites.
- Não coletamos dados de saúde ou biológicos por meio deste site.

---

## Arweave e Dados On-Chain

Os dados armazenados na rede Arweave via contratos inteligentes BSP são **permanentes e públicos por design**. O protocolo é construído para que apenas a chave criptográfica do titular do BEO possa autorizar o acesso — mas a existência de transações on-chain é publicamente visível.

Antes de enviar qualquer dado por meio de um aplicativo compatível com BSP, certifique-se de entender a natureza permanente da rede Arweave.
