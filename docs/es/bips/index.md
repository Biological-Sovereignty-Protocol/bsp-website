# Propuestas de Mejora BSP (BIPs)

BSP evoluciona a través de un proceso público de mejora. Cualquiera puede proponer cambios al protocolo.

## Cómo Enviar un BIP

1. Haz un fork de [bsp-spec en GitHub](https://github.com/Biological-Sovereignty-Protocol/bsp-spec)
2. Copia `bip/BIP-0000-template.md` a `bip/BIP-XXXX-tu-titulo.md`
3. Completa el template en su totalidad
4. Abre un Pull Request

El PR abre un período de revisión pública. Consulta [Gobernanza](../specification/governance) para el proceso completo.

## Categorías de BIP

| Categoría | Descripción |
|---|---|
| **BSP-BIP-TAXONOMY** | Agregar o modificar códigos de biomarcadores |
| **BSP-BIP-SCHEMA** | Cambios al schema de BEO, IEO o BioRecord |
| **BSP-BIP-EXCHANGE** | Cambios al Protocolo de Intercambio |
| **BSP-BIP-GOVERNANCE** | Cambios al proceso de gobernanza |
| **BSP-BIP-INFRA** | Actualizaciones de contratos inteligentes |
